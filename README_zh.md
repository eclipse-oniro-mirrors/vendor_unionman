# vendor_unionman

## 介绍

该仓库托管Unionman产品：Unionpi Tiger（Amlogic A311D）系列OpenHarmony硬件。

## 软件架构

支持基于Amlogic A311D芯片进行开发。

代码路径：

```
vendor/unionman/         // vendor_unionman 仓的仓库路径
├── a311d                // Unionpi Tiger开发板
└── ...

```
